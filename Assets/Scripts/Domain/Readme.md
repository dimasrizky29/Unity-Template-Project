# Domain Folder

Folder ini berisi **The Business Logic**: jantung dari game. Di sini mendefinisikan "Kontrak" (interface, abstraksi) dan "Data" (model, entitas) yang menjadi inti dari logika aplikasi/game. Semua skrip di folder ini fokus pada aturan bisnis dan struktur data, tanpa bergantung pada detail implementasi atau teknologi spesifik.
